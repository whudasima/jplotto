宝くじ公式サイトキャンペーン規約

第1条 本規約の位置付け

  1. 「宝くじ公式サイトキャンペーン規約」（以下「本規約」という。）は、宝くじを発売する全国都道府県及び全指定都市で構成する協議会（以下、「当協議会」という。）が提供する宝くじ公式サイト（以下「本サイト」という。）で取扱う各種キャンペーンにおいて、共通する事項について規定するものです。
  2. 各種キャンペーンにおいて、本規約に加えて個別に詳細なキャンペーン条件を定める場合（以下「個別規約」という。）であって、本規約と個別規約の内容が相違する場合は、個別規約を優先するものとします。
  3. 当協議会は、本規約の各条項その他の条件を、民法第548条の4の規定により、事業環境の変化その他相当の事由があると認められる場合には、変更内容を記載した店頭ポスター掲示または本サイト掲載による公表その他相当の方法で周知することにより、変更できるものとします。
  4. 前項の変更は、公表等の際に定める適用開始日から適用されるものとします。

第2条 本サイトにおけるキャンペーン対象者

  1. 本サイトにおけるキャンペーン対象者は、本サイトの会員であることを条件とします。
  2. 本規約における「会員」とは、「宝くじ会員規約」に規定する会員をさすものとします。
  3. 法人、満20歳未満の方および外国に居住されている方は、キャンペーンに応募することはできません。

第3条 本サイトにおけるキャンペーンの当せん

本サイトにおけるキャンペーンの当せん内容には、以下のものがあります。

  1. 宝くじポイントの付与
  2. 賞品の提供（デジタルコンテンツなど、物理的な送付対象とならないものを含む。）
  3. その他当協議会が適切と判断するもの

第4条 宝くじポイントの付与

  1. 当協議会は、会員が個別規約に定めるキャンペーンの条件を満たした場合、一定の期間経過後に宝くじポイントを付与します。
  2. キャンペーンにより付与される宝くじポイントは、宝くじ購入によって付与される宝くじポイントより有効期間が短くなる場合があります。キャンペーンにより付与される宝くじポイントの有効期限は、個別規約にて定めるものとします。
  3. その他ポイントに関する事項で、本規約ならびに個別規約で定めがない事項は、宝くじ会員規約の定めに従うものとします。

第5条 賞品の送付

  1. 賞品（デジタルコンテンツなど、送付対象にならないものを除く。）は、指定がない場合を除き、会員情報として登録された住所に送付します。
  2. 会員が、会員情報として登録された住所以外に賞品の送付を希望する場合、個別規約で制限している場合を除き、会員が希望する住所に賞品を送付します。
  3. 賞品の送付先は、日本国内に限定します。
  4. 賞品の送付先は、応募後に変更することはできません。
  5. 賞品の配送中または提供後に遅延、紛失、盗難、損害、破損等の事故が生じた場合は、当該事故が当協議会の責任による場合を除き、当協議会は一切責任を負いません。

第6条 抽せん、当せん

  1. 個別規約において抽せんを実施することを規定している場合、キャンペーンの期限後に厳正な抽せんを行います。
  2. 当せんの発表は、当せん者本人への宝くじポイントの付与、賞品の発送（デジタルコンテンツなど送付対象とならないものは権利の付与または通知。）をもって発表にかえます。なお、宝くじポイントを付与した場合は、会員情報として登録されたメールアドレス宛の電子メールの送信、本サイト内の会員専用のページへの表示、その他当協議会が適当と判断する方法によって通知等を行います。
  3. 抽せんを実施する場合は、1回のキャンペーンにおいて、同一人が重複して当せんすることはありません。

第7条 当せんの権利・賞品の第三者への譲渡

  1. 当せんの権利および賞品については、第三者への譲渡や換金、転売等は一切禁止します。
  2. 前項の違反が判明した場合、当せんを無効とする場合があります。

第8条 個人情報の取扱い

当協議会は、キャンペーンに関して取得する個人情報を、当協議会のプライバシーポリシー（https://www.takarakuji-official.jp/privacypolicy.html）に従い、適切に取扱います。

第9条 キャンペーン内容等に関する変更への同意

キャンペーンの内容については、やむを得ず以下の事項が発生する場合があります。会員は、以下の事項が発生することについて、あらかじめ同意するものとします。

  1. 諸事情により、一部賞品の内容が変更されること
  2. 賞品の実物と、本サイトでの表現に色味などの差が生じること
  3. 諸事情により、賞品の送付が予定から遅れる場合があること
  4. 諸事情により、個別規約に定められた応募期限の到来前に応募受付を終了する場合があること
  5. その他、当協議会が必要と判断することにもとづく変更等

第10条 その他

本規約ならびに個別規約に定めがない事項は、宝くじ会員規約の関係条文に従うものとします。
